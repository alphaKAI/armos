Why use D?

We choose to implement armos in D programming language. This is because of the following reasons. First, the language system has terse and high orthogonality. Second, the compilation is more faster than a speed of C++. Because of that, we can repeat trial and error. Furthermore, the processing speed has performance comparable to C++.

armosはD言語により実装されています．簡潔で直交性のある言語システム，圧倒的コンパイル速度，そしてバイナリによるC++にも匹敵する実行時の速度．これらの理由からD言語を選択しました．
