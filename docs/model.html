armos.graphics.model

class Model;
3Dモデルを読み込み，描画するclassです．

void load(string pathInDataDir);
モデルを読み込みます．

読み込み時にmeshはmaterial毎に分割されます．

size_t numMeshes();
読み込まれたmeshの数を返します．

size_t numMaterials();
読み込まれたmaterialの数を返します．

void draw(armos.graphics.PolyRenderMode renderMode);
modelを描画します．

Params:
armos.graphics.PolyRenderMode renderMode 面，線，点のどれを描画するか指定します．

void drawWireFrame();
modelをワイヤフレームで描画します．

void drawVertices();
modelの頂点を点で描画します．

void drawFill();
meshの面を塗りつぶして描画します．

class AssimpModelLoader;
Assimpによりmodelの読み込みを行います．

Model load(string pathInDataDir);
modelを読み込みそれを返します．

void clear();
読み込んだモデルを削除します．
