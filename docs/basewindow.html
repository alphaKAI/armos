armos.app.basewindow

interface Window;
armosで用いるWindowsの雛形となるinterfaceです．新たにWindowを実装する際はこのinterfaceを継承することでrunnerから実行できます．

abstract armos.events.CoreEvents events();
Windowsが実行するイベントを表すプロパティです．

abstract armos.math.Vector2i size();
サイズを返すプロパティです．

abstract void pollEvents();
イベントが発生している場合，登録されたイベントを実行します

abstract void update();
Windowを更新します．

abstract void close();
Windowを閉じます．

abstract bool shouldClose();
Windowがフレームの最後に閉じる場合trueになります．

abstract float aspect();
Windowのアスペクト比を表します

abstract string name();
Windowのタイトル文字列のプロパティです．

abstract void name(string str);
Windowのタイトル文字列のプロパティです．

class SDLWindow: armos.app.basewindow.Window;
SDLを利用したWindowです．

Deprecated:
GLFWWindowの使用が推奨されています．

class GLFWWindow: armos.app.basewindow.Window;
GLFWを利用したWindowです．armosではデフォルトでこのclassを元にWindowが生成されます．

this(ref armos.app.BaseApp apprication);
Params:
armos.app.BaseApp apprication Windowとひも付けされるアプリケーションです．

armos.math.Vector2i size();
Windowのサイズを返します．

void pollEvents();
イベントが発生している場合，登録されたイベントを実行します

void update();
Windowを更新します．

void close();
Windowを閉じます．

armos.app.Window currentWindow();
現在のWindowを返す関数です．

armos.math.Vector2i windowSize();
現在のWindowの大きさを返す関数です．

float windowAspect();
現在のWindowのアスペクト比を返す関数です．
