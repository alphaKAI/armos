armos.graphics.camera

class Camera;
Cameraを表すClassです．Cameraで写したい処理をbegin()とend()の間に記述します．

armos.math.Matrix4f projectionMatrix();
projectionMatrixを取得します．

armos.math.Vector3f position;
Cameraの位置を表します．

armos.math.Vector3f target;
Cameraが映す対象の位置を表します．

armos.math.Vector3f up;
Cameraの方向を表します．

double fov;
Cameraの視野角を表します．単位はdegreeです．

double nearDist;
描画を行う最短距離です．

double farDist;
描画を行う最長距離です．

void begin();
Cameraで表示する処理を開始します．

void end();
Cameraで表示する処理を終了します．
